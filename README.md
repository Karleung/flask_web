### git项目提交：

[init] 初始化

[feat] 新特性、新功能

[modify] 修改功能

[delete] 删除功能或文件

[fix] 修复问题或解决冲突

[docs] 修改文档

[opt] 功能优化、代码重构

[build] 改变构建流程、新增依赖、工具

[style] 仅修改代码格式（空格、缩进、注释）

[test] 测试用例新增、修改

[revert] 回滚上一个版本
